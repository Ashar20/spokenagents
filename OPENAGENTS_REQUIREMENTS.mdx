# Open Agents Prize Requirements Checklist

Source: [ETHGlobal Open Agents Prizes](https://ethglobal.com/events/openagents/prizes#0g)

## Universal Submission Checklist

- [ ] Project name and short description
- [ ] Contract deployment addresses
- [ ] Public GitHub repo with README and setup instructions
- [ ] Demo video (under 3 minutes)
- [ ] Live demo link
- [ ] Notes on protocol features/SDKs used
- [ ] Team member names and contact info (Telegram + X)

## 0G

### Tracks

- Best Agent Framework, Tooling & Core Extensions
- Best Autonomous Agents, Swarms & iNFT Innovations

### Required

- [ ] Submission includes universal checklist items
- [ ] If framework/tooling track: framework-level work (extensions/forks/new framework/tooling)
- [ ] At least one working example agent built with your framework/tooling
- [ ] If swarm project: clear communication/coordination explanation
- [ ] If iNFT project: explorer link + proof intelligence/memory is embedded

### Links

- [Builder Hub](https://build.0g.ai)
- [Telegram Support](https://t.me/+mQmldXXVBGpkODU1)

## Gensyn (AXL)

### Track

- Best Application of Agent eXchange Layer (AXL)

### Required

- [ ] Must use AXL for inter-agent or inter-node communication
- [ ] Must demonstrate communication across separate AXL nodes (not only in-process)
- [ ] Project built during hackathon
- [ ] Clear docs and working examples

### Judging Focus

- Depth of AXL integration
- Code quality
- Documentation clarity
- Working example/demo quality

### Links

- [AXL Documentation](https://docs.gensyn.ai/tech/agent-exchange-layer)
- [AXL GitHub Repo](https://github.com/gensyn-ai/axl)
- [Collaborative Autoresearch Demo](https://github.com/gensyn-ai/collaborative-autoresearch-demo)

## Final Pre-Submit Sweep

- [ ] Repo is public and build instructions are reproducible
- [ ] Demo video is under 3 minutes
- [ ] All sponsor-specific mandatory files are present (example: `FEEDBACK.md` for Uniswap)
- [ ] Sponsor-specific proof points are included (swarm/iNFT/ENS utility/etc.)
- [ ] Contact details are included and correct
